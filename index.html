2. Adding Images

NOTE: image element is self-containing and empty, it doesn’t wrap any other content and it exists as a single tag. a src attribute and value must be included to specify the source of the image. The src attribute value is a URL, relative to the server where a website is hosted.

    WAYS TO ADJUST IMAGE SIZE:
  1. adjust width by percentage, the image with adjust size along with the browser screen size
  2. adjust by width: the image size is fixed and will keep its proportion
  3. adjust by both height and width

Image formats: png for logos and icons, jpg for photography,better quality
